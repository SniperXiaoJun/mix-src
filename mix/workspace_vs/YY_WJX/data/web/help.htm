《学OpenGL编3D游戏》提供完整的源程序

帮助 ● 说明

    这是《学OpenGL编3D游戏》的教学方案。采用循序渐进的方法，从建立OpenGL图形环境入手，讲解3D基本图形、构图原理；从引入摄像机，建立天空、山地、树木，到3D模型使用和3D动画的显示；完整地展示了游戏的编写过程。
    《学OpenGL编3D游戏》的教学课件。用多媒体的表现手法将学习过程完全显示在你面前，使用者可以随时查看所选章节的知识要点提示，可以观看程序的制作过程和效果，也可以马上进入到VC编辑器对范例程序修修改改，在实践中加深对知识的理解；还可以进入到网上论坛和朋友们讨论学习心得。

    每章的示范程序都有两种方式。一是在本程序的窗口中出现的，另外在运行目录下有各自的单独运行程序。这里提供的各章节源程序为单独运行的SDK程序，你可以随时调入到VC中改写。
    希望本教材可以带你愉快地进入编写3D游戏的领域，实现你的梦想。

   运行程序可能出现的问题，解决方法


操作界面   点击放大

   OpenGL程序运行太慢
   OpenGL画面右下角的刷新指标小于5帧/秒。原因: Win2000或WinXP在安装时，系统自动安装默认的显卡驱动程序，多数默认的显卡驱动程序是不完全的版本，对OpenGL的支持不够(微软有意的?)。所以应该重新安装随机带的厂商提供的显卡驱动程序。当然，你的机器太差，特别是显卡太老，就没办法了。就像作者的一台C400的东芝笔记本电脑，显卡内存只2.5Mb,并不支持3D加速...就没法了。
   源程序不能编译
  1、你的机器没安装 Visual C++，当然不行了。本教程所有程序是在Visual C++ 6.0上编写的。
  2、你可能是直接在光盘上运行吧。光盘是只读的，编译程序时大量的过程文件不能建立，当然就不能编译了。请运行光盘根目录下的SETUP.EXE将程序安装到你的硬盘上吧。
